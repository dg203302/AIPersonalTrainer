Guía del Plan de Ejercicio

Paso a paso y funciones

Guía

Acá tenés una guía completa para generar, ver y gestionar tu plan semanal. El plan se arma con IA usando tu perfil (edad, altura, peso actual y objetivo) + las opciones que elijas.

Requisito

Tené tu perfil completo para mejores resultados.

Qué vas a ver

Solo se muestran los días que tienen ejercicios asignados.

1) Generar el plan

  1. Entrá a Plan de Entreno (el botón del footer).
  2. Presioná el botón principal de arriba (Plan de Entreno / Generar).
  3. Se abre una ventana con opciones: completalas y tocá Generar.
  4. Cuando termine, el plan aparece en el recuadro Plan de Ejercicio.

2) Opciones disponibles (qué hace cada una)

  • ¿Dónde entrenás? Elegí casa o gimnasio para que el plan use ejercicios y equipo acordes.
  • ¿Qué priorizás? Elegí pérdida de grasa o ganancia muscular para ajustar el enfoque y la progresión.
  • Días de la semana Marcá L, M, X, J, V, S, D: esos serán tus días de entrenamiento.
  • Ejercicios disponibles (índice) (opcional):
    • Si activás “Quiero elegir ejercicios” y seleccionás ejercicios, el plan se crea usando solo esos ejercicios (no agrega otros).
    • Si no seleccionás nada, la IA elige automáticamente del índice disponible.

Tip: si seleccionás muy pocos ejercicios para muchos días, algunos pueden repetirse para completar el plan.

3) Cómo leer el plan

  • Día: cada “tarjeta” corresponde a un día (por ejemplo Lunes).
  • Ejercicio: lista de ejercicios del día.
  • Series: cuántas tandas hacés.
  • Repeticiones: cuántas reps por serie (o rango sugerido).
  • Descanso: tiempo recomendado entre series.
  • Progresión: sugerencia para ir subiendo carga/reps/series con el tiempo.

4) Navegación por días (scroll por “pantallas”)

  • En celular, deslizá arriba/abajo sobre el recuadro del plan para pasar de un día al siguiente.
  • En PC, usá la rueda del mouse sobre el recuadro para avanzar/retroceder día por día.
  • Si un día tiene muchos ejercicios, podés scrollear dentro del día para ver todo el contenido.

5) Ver detalle por día

  • Tocá/clickeá el encabezado del día (por ejemplo “Martes”).
  • Se abre un modal con el detalle de ese día (series, reps y descanso por ejercicio).

6) Gestionar tu plan

  • Botón principal: genera el plan si no tenés uno; si ya existe, puede actualizar/refrescar según el estado.
  • Regenerar Plan: crea un plan nuevo basado en tu configuración previa.
    • Te pide confirmación.
    • Al aceptar, elimina el plan actual.
    • Abre el modal con opciones precompletadas (lugar, objetivo, días y, si aplica, ejercicios) para que ajustes y generes de nuevo.
  • Eliminar Plan: borra el plan actual para empezar desde cero.

7) Solución rápida de problemas

  • No aparece el plan: revisá conexión, recargá la página y verificá que tu perfil esté completo.
  • El plan no cambia: al regenerar, confirmá la acción y volvé a generar con nuevas opciones.
  • Días “faltantes”: solo se muestran días con ejercicios asignados.

Atajo: desde el encabezado (arriba a la derecha) tocá Perfil para editar tus datos.
