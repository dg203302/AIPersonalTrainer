Plan de Entreno

Acá vas a poder generar un plan de entrenamiento personalizado (rutina semanal) con ejercicios, series/repeticiones, descansos y una progresión sugerida. Para armarlo se tiene en cuenta dónde entrenás (en casa o en gimnasio), tu objetivo principal (priorizar pérdida de grasa o ganancia muscular) y tus datos básicos (edad, altura, peso actual y peso objetivo).

Ayuda Guía

Desde esta sección podés generar un plan semanal con ejercicios, series/repeticiones, descansos y una progresión sugerida. El plan se arma con IA usando tu perfil (edad, altura, peso actual y objetivo) + las opciones que elijas.

1) Generar el plan

  1. Presioná el botón principal de Plan de Entreno (el que está arriba del plan).
  2. Se abre una ventana con opciones. Completalas y tocá Generar.

2) Cómo elegir cada opción

  • ¿Dónde entrenás? Elegí casa o gimnasio para que el plan use ejercicios y equipo acordes.
  • ¿Qué priorizás? Elegí pérdida de grasa o ganancia muscular para ajustar el enfoque y la progresión.
  • Días de la semana Tocá las letras (L, M, X, J, V, S, D) para marcar qué días vas a entrenar.
  • Ejercicios disponibles (índice) (opcional) Activá “Quiero elegir ejercicios” y marcá ejercicios preferidos. Si no marcás nada, la IA elige automáticamente.

3) Ver el plan y gestionarlo

  • Cuando termine de generarse, el plan aparece debajo en Plan de Ejercicio. Si es largo, podés scrollear dentro del recuadro.
  • Ver detalle por día: tocá el encabezado de un día (por ejemplo “Lunes”, “Martes”, etc.) para abrir un modal con el detalle de cada ejercicio de ese día (series, repeticiones y descanso).
  • Si querés volver a generar, repetí el proceso y elegí nuevas opciones.
  • Regenerar Plan: te permite crear un plan nuevo basado en la configuración previa. Al tocarlo, se te pedirá confirmación; si aceptás, se elimina el plan actual y se abre el modal con las opciones precompletadas (lugar, objetivo, días y, si aplica, ejercicios) para que ajustes lo que quieras y generes nuevamente.
  • Si aparece el botón Eliminar Plan, podés borrar el plan actual para empezar de cero.

4) Cómo moverte por la interfaz

  • Encabezado: arriba a la derecha ves tu nombre y avatar. Tocá el ícono de Perfil para ir a tus datos.
  • Botón principal: el botón grande de arriba sirve para generar tu plan (si no tenés uno) o refrescar/actualizar el plan (si ya existe).
  • Regenerar Plan: aparece cuando ya tenés un plan. Te guía para crear uno nuevo usando la configuración previa.
  • Eliminar Plan: borra el plan actual. Útil si querés empezar desde cero.
  • Zona del plan: el recuadro de Plan de Ejercicio puede tener scroll propio. En celular, deslizá dentro del recuadro para leer todo.
  • Ayuda: esta guía se puede abrir/cerrar tocando el título “Ayuda: cómo generar y ver tu plan”.

Tip: si no ves cambios, revisá tu conexión y que tu perfil esté completo.
